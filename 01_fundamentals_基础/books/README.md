# Typescript

1. [Effective TypeScript](https://rayadaschn.github.io/effective-typeScript-2nd-edition-zh/)


根据目前的行业共识，以下资源是最高效的：

官方手册 (Handbook)： 始终是第一参考。2026 年的 TS 已经更新了许多性能优化特性，务必关注最新的 Template Literal Types。

经典书籍： 《Effective TypeScript》（Dan Vanderkam 著）。它不仅教你语法，更教你 83 条最佳实践（例如：为什么少用 any，如何设计最小化类型）。

实战挑战： Type Challenges (GitHub)。通过“刷题”的方式掌握类型级编程，这是从初级跨越到高级的必经之路。
